# 🌐 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and Bootstrap**. This project showcases a professional portfolio with a clean UI, smooth animations, and responsive design.

---

## 📖 Project Overview

This portfolio website is designed to present personal and professional information in an attractive way. It includes essential sections such as About, Education, Skills, Certifications, Contact Information, and Footer. The website is fully responsive and works well on desktops, tablets, and mobile devices.

---

- 🎨 Modern and Bright UI Design
- 📱 Fully Responsive Layout
