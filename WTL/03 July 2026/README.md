# ✨ Modern To-Do List

A modern, stylish, and animated To-Do List web application built using **HTML, CSS, and JavaScript**. The project follows an Apple-inspired design philosophy with glassmorphism effects, vibrant gradients, smooth animations, and a user-friendly interface.

---

## 📌 Project Overview

The Apple Modern To-Do List is a lightweight task management application that helps users organize their daily activities efficiently.

The application provides a clean and visually appealing interface where users can:

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Manage daily activities easily
- Experience smooth animations and modern UI effects

The project is built entirely with frontend technologies and does not require any backend or database.

## ▶️ How to Use

### Adding a Task

1. Type a task in the input field.
2. Click the Add button.
3. The task will appear in the list.

### Completing a Task

1. Click the ✓ button.
2. The task will be marked as completed.

### Deleting a Task

1. Click the 🗑 button.
2. The task will be removed from the list.
